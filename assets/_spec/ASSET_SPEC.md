# アセット仕様書（デザイナー向け）

## 命名規則

ファイル名 = 各JSONで参照する `sprite` / `asset` ID と一致させる。

- 例: `enemies.json` に `"sprite": "slime"` → `assets/sprites/enemies/slime.png`
- 例: カットシーンに `"asset": "battle"` → `assets/bgm/battle.mp3`

## スプライト仕様

| カテゴリ | フォーマット | サイズ | フレーム規約 |
|---|---|---|---|
| キャラスプライト | PNG (透過) | 48x64 px | 3列x4行（下/左/右/上 x 3歩行フレーム） |
| NPCスプライト | PNG (透過) | 48x64 px | 同上（静止の場合は16x16の1フレームでも可） |
| 敵スプライト | PNG (透過) | 最大 64x64 px | 1枚絵（正面向き） |
| タイルセット | PNG | 横幅自由 x 高さ自由 | 16x16 グリッド配置。タイルIDは左上から右へ 0,1,2... |
| UIパーツ | PNG (透過) | 個別指定 | 9-slice対応の場合は4隅+辺+中央 |

## オーディオ仕様

| カテゴリ | フォーマット | 備考 |
|---|---|---|
| BGM | MP3 + OGG | ループ: ファイル先頭〜末尾でシームレスループ |
| SE | MP3 | 1秒以内推奨 |

## キャラスプライトシート詳細

```
┌─────┬─────┬─────┐
│ 下0 │ 下1 │ 下2 │  行0: 下向き
├─────┼─────┼─────┤
│ 左0 │ 左1 │ 左2 │  行1: 左向き
├─────┼─────┼─────┤
│ 右0 │ 右1 │ 右2 │  行2: 右向き
├─────┼─────┼─────┤
│ 上0 │ 上1 │ 上2 │  行3: 上向き
└─────┴─────┴─────┘
各フレーム: 16x16 px
```

歩行アニメーション: フレーム0→1→0→2 の繰り返し
